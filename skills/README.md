# Skills Compartilhadas

Esta pasta contém apenas skills canônicas compartilhadas pelo hub.

## Regra

- A fonte de verdade das skills compartilhadas fica aqui.
- Provedores podem manter bootstrap local, mas não devem manter aqui cópias
  divergentes como fonte canônica.
- O índice de resolução fica em `manifests/skills.index.yaml`.
