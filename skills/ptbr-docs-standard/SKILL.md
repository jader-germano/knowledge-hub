# ptbr-docs-standard

Use esta skill quando a tarefa envolver documentação compartilhada do workspace.

## Regra De Idioma

- Documentação do repositório e do hub em `pt-BR`
- Código, testes, nomes de tipos, nomes de funções, contratos e exemplos
  técnicos em inglês
- UI e apresentação em `pt-BR` apenas quando o idioma selecionado for `pt-BR`

## Aplicação

- Normalize breadcrumbs, índices, guias e texto explicativo em `pt-BR`
- Preserve termos técnicos em inglês quando forem nomes próprios, APIs,
  símbolos ou contratos
- Não use a skill para traduzir código

## Ownership

- Esta skill é compartilhada pelo hub
- Qualquer bootstrap de provedor deve apontar para esta versão
