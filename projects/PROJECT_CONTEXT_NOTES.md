# Project Context Notes

Os arquivos `projects/<repo>/PROJECT_CONTEXT.md` são resumos operacionais.

Eles existem para:

- reduzir o custo de bootstrap das LLMs
- dar visão rápida de nicho, stack e sinal recente
- apontar sempre para o repositório real em `/Users/philipegermano/code`

Exceção:

- contextos internos de estratégia podem apontar também para fontes documentais
  fora de `/Users/philipegermano/code`, desde que o hub continue sendo o espelho
  canônico usado pelas LLMs

Eles não substituem:

- README do repositório real
- AGENTS.md do repositório real
- documentação de produto ou arquitetura mantida no próprio repo
