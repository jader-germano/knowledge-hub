# Shared LLM Context Template

Template canônico para o bootstrap transversal de providers no `Knowledge Hub`.

## Consulta Obrigatória

1. `README.md`
2. `MCP_SETUP.md`
3. `Projects/Knowledge-Hub.md`
4. `memory/PI_MEMORY.md`
5. `memory/AGENTS.md`

## Regras Operacionais

- tratar `origin` como remoto canônico do `Knowledge Hub`
- tratar este repositório privado como fonte de verdade da documentação
  transversal do workspace
- registrar mudanças transversais aqui antes de espalhar bootstrap para notas
  locais, clientes específicos ou runtimes de provider
- manter apenas instruções específicas de provider nos arquivos
  `llms/<PROVIDER>.md`; o conteúdo compartilhado deve viver neste template
