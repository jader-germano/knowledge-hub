Design Principles
Claude.app
  • Warmth & approachability — off-white tones, generous whitespace
  • Typography-first — clear hierarchy, readable at all sizes
  • Minimal chrome — content takes center stage
  • Conversational rhythm — natural message bubbles, human-first layout
  • Trust through clarity — no hidden state, predictable interactions
Codex.app
  • Code-native — dark theme mirrors IDE conventions
  • Power-user density — compact layout, keyboard-first UX
  • Agentic feedback — real-time diffs, terminal output, status badges
  • Multi-panel layout — file tree, editor, and agent side-by-side
  • Automation trust — clear run/commit/rollback affordances
